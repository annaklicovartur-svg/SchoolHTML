1️⃣ Структура документа
Тег	Что делает	Пример

<!DOCTYPE html>	Объявляет тип документа (HTML5)	в самом начале

<html>	Корневой элемент всей страницы	оборачивает всё

<head>	Мета-информация (не видна на странице)	заголовок, кодировка

<title>	Заголовок на вкладке браузера	<title>Мой сайт</title>

<meta charset="UTF-8">	Устанавливает кодировку (чтобы не было непонятных символов)	в <head>

<body>	Всё, что видит пользователь	оборачивает контент

2️⃣ Текстовые теги
Тег	Что делает	Визуально

<h1> - <h6>	Заголовки (от самого большого до маленького)	Жирный, размер разный

<p>	Параграф (абзац текста)	Отступ сверху и снизу

<b>	Жирный текст (визуально)	жирный

<strong>	Жирный + важный для поисковиков	жирный

<i>	Курсив (визуально)	курсив

<em>	Курсив + логическое ударение	курсив

<sub>	Подстрочный текст	H~2~O

<sup>	Надстрочный текст	E=mc^2^

<br>	Перенос строки	переводит на новую строку

<hr>	Горизонтальная линия-разделитель	линия во всю ширину

<pre>	Сохраняет пробелы и переносы	как в коде

<code>	Для отображения кода	моноширный шрифт

<abbr>	Аббревиатура с подсказкой	при наведении всплывает текст

3️⃣ Списки
Тег	Что делает	Вид

<ul>	Ненумерованный список	• пункт • пункт
<ol>	Нумерованный список	1. пункт 2. пункт
<li>	Элемент списка (внутри ul/ol)	сам пункт

4️⃣ Ссылки и навигация
Тег/Атрибут	Что делает	Пример

<a href="url">	Создаёт ссылку	<a href="https://google.com">Google</a>

href="#id"	Ссылка на якорь внутри страницы	<a href="#footer">Вниз</a>

href="mailto:email"	Ссылка на отправку письма	<a href="mailto:test@mail.ru">Написать</a>

target="_blank"	Открыть ссылку в новой вкладке	добавляется к <a>

name="имя"	Создаёт якорь (устаревший, но работает)	<a name="footer"></a>

title="текст"	Всплывающая подсказка при наведении	у любого тега


5️⃣ Изображения
Тег/Атрибут	Что делает

<img src="путь">	Вставляет картинку
src="data:image/..."	Картинка в кодировке base64 (прямо в HTML)
alt="текст"	Текст, если картинка не загрузилась
width="..."	Ширина картинки
title="..."	Подсказка при наведении


6️⃣ Таблицы
Тег	Что делает

<table>	Контейнер для таблицы
border="число"	Толщина рамки (устаревший, но работает)
width="..."	Ширина таблицы
<thead>	Шапка таблицы
<tbody>	Тело таблицы (основные данные)
<tfoot>	Подвал таблицы (итоги)
<tr>	Строка таблицы
<th>	Ячейка-заголовок (жирный, по центру)
<td>	Обычная ячейка с данными
align="center"	Выравнивание содержимого (устаревший)


7️⃣ Формы и ввод данных
Тег/Атрибут	Что делает

<form>	Контейнер для формы

action="url"	Куда отправить данные

method="get/post"	Способ отправки

<input>	Поле ввода (текст, пароль, кнопка и др.)

type="text"	Текстовое поле

placeholder="..."	Подсказка внутри поля

maxlength="..."	Максимум символов

value="..."	Значение по умолчанию

disabled	Поле неактивно

readonly	Только для чтения

hidden	Скрытое поле

<textarea>	Большое поле для текста (с переносами)

<button>	Кнопка

type="reset"	Сбрасывает форму

<select>	Выпадающий список

<option>	Элемент выпадающего списка

selected	Элемент выбран по умолчанию

disabled	Элемент недоступен


8️⃣ Дополнительные теги
Тег	Что делает

<progress>	Индикатор загрузки (прогресс-бар)
<details>	Раскрывающийся блок
<summary>	Заголовок раскрывающегося блока
<div>	Универсальный блочный контейнер
<span>	Универсальный строчный контейнер


9️⃣ Атрибуты-модификаторы
Атрибут	К каким тегам	Что делает

type="I"	<ol>	Римские цифры для нумерации

start="2"	<ol>	Начать нумерацию с 2

colspan	<td>, <th>	Объединяет ячейки по горизонтали

🎯 Шпаргалка "Какой тег для чего":

Хочу сделать	Использую
Заголовок	<h1>
Абзац	<p>
Жирный текст	<b> или <strong>
Курсив	<i> или <em>
Ссылку	<a href="...">
Картинку	<img src="...">
Список	<ul> + <li> или <ol> + <li>
Таблицу	<table> + <tr> + <td>
Форму	<form> + <input>
Перенос строки	<br>
Линию-разделитель	<hr>
Код	<pre> или <code>
Подсказку	title="..." или <abbr>
